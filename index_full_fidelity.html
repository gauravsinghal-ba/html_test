How to use NotebookLM?

The official Adyogi workflow for creating L&D Masterclass material.

Adyogi Training v2.0
Step 1

Create a Notebook

  • Log into Notebook LM via individual Adyogi Email.
  • Click Create a New Notebook.
Step 2

Add Sources (Max 5)

  • Click on Add sources.
  • Upload relevant training documents like webinar, confluence, decks etc.
  • Search the web for additional sources, using a prompt:
Reference Prompt

"Share the most relevant sources in the last 6/12 months on the topic 'Scaling during GIF on Amazon' prioritise sources from Amazon."

  • View, Audit & Import the most relevant sources.
Step 3

Create an Instruction Document

Use the following Gemini Prompt to get an instruction document (Helps describe the flow, goal & tone of the deck).

Gemini Prompt

"Act as an Industry Expert & Create a structured instruction document covering the following four sections: 1) Introduction: Topic.. 2).... 3)......., aim to Move beyond basic theory to provide a masterclass on [Topic] Ensure the tone & style: Professional, Collaborative etc."

  • Add the Instruction document as a source in the notebook.
Step 4

Generate the deck in Studio

  • Click on the edit sign over the slide deck option in studio.
  • Click Detailed Deck.
  • Choose English.
  • Enter the prompt under: Describe the slide deck you want to create.
Final Studio Prompt

"Create a comprehensive master class Deck as an Amazon PPC Strategist. THE GOAL is To train a team of performance marketers on the End-to-End strategy for winning the Great Indian Festival (GIF), using the attached sources. Leverage the instructions attached in the sources to define the flow of the video. Ensure the video is for L&D and is detailed."

Best Practices

Source Guidelines
  • Restrict sources to a maximum of 5 (~3 internal, ~2 external).
  • Use Fast Research for broad topics, Deep Research for specific ones.
  • In case you wish to use a part of a comprehensive source, create a note of that part and use it as a source instead of using the comprehensive source to include everything.
  • Audit all sources to figure the most relevant sources for your notebook to give meaningful results.
Instruction Essentials

Add an instruction document (preferably Google Doc or Adyogi Template) outlining:

Flow of material
Tone & Style
Specific Details
Design/Format
Visuals & Graphs

If you wish to include graphs and pictures, ensure sources include these or add a file from Canva.

Final Prompt Checklist

  • Topic Name
  • Define Role (Expert/Strategist)
  • Content Length
  • Audience Definition
  • Define Objective
  • Leverage Instructions Doc
